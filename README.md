This project analyzes workforce data to uncover structural drivers of attrition, performance misalignment, and retention risk.

Focus Areas
- Attrition segmentation by tenure and performance
- Engagement vs retention analysis
- Manager- and department-level churn patterns

Key Findings

- Engagement alone does not guarantee retention
- Structural churn clusters around specific teams and managers
